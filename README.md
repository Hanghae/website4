
# HSJ Portfolio — Vite + React + Tailwind

## Quick start
```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build
```bash
npm run build
npm run preview
```

## Deploy (Vercel)
- Push this folder to GitHub
- On vercel.com: Add New → Project → Import repo → Deploy (Build: `npm run build`, Output: `dist`)

## Notes
- Replace the hero video `src` in `App.tsx` with your 6–8s WebM (muted).
- Replace the placeholder thumbnails/previews with your actual media.
- Edit `index.html` title/OG tags for SEO.
